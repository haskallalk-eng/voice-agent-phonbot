So klingt Phonbot am Telefon

Phonbot: „Oh je, das klingt dringend. Ich nehme deinen Notdienst-Fall sofort auf. Deine Adresse bitte?"

📞 Kunde: „Hauptstraße 12 in Berlin."

Phonbot: „Danke. Wie heißt du und unter welcher Nummer kann der Techniker dich zurückrufen?"

Häufige Fragen

Erkennt Phonbot den Unterschied zwischen Notfall und regulärem Termin?

Ja. Keywords wie "sofort", "Wasserschaden", "brennt" lösen automatisch Priorität-hoch aus. Du kannst im Agent Builder deine eigenen Notfall-Regeln ergänzen.

Kann ich unterwegs auf Anfragen reagieren?

Ja. Alle Tickets kommen im Dashboard an, zusätzlich per E-Mail-Benachrichtigung. Mobile-App kommt 2026.

Versteht Phonbot Handwerks-Fachbegriffe (Thermostat, Siphon, etc.)?

Ja. Das LLM kennt deutsche Handwerks-Terminologie und kann bei Unklarheit gezielt nachfragen.

Was wenn mehrere Kunden gleichzeitig anrufen?

Phonbot nimmt mehrere Anrufe parallel an (eigener Agent pro Gespräch). Niemand hängt in der Warteschleife.
