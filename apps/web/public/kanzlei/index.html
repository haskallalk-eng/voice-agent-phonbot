Häufige Fragen

Ist das mit der anwaltlichen Schweigepflicht vereinbar?

Ja. Phonbot ist Datenverarbeiter im Sinne der DSGVO mit AV-Vertrag. Erste Mandats-Details werden verschlüsselt gespeichert, der finale Mandatsvertrag bleibt wie gewohnt direkter Anwalt-Mandant-Bezug.

Kann Phonbot zwischen verschiedenen Rechtsgebieten unterscheiden?

Ja. Im Agent Builder legst du deine Spezialisierungen fest (z.B. IT-Recht, Arbeitsrecht). Phonbot qualifiziert eingehende Fälle entsprechend.

Was bei Fristen-Fällen (z.B. Klagefrist)?

Fristen-relevante Keywords lösen Prioritäts-Ticket aus. Anwalt bekommt sofortige Push-Nachricht für dringliche Kontaktaufnahme.

Können wir eigene Fragen stellen lassen (z.B. zur RSV)?

Ja. Im Agent Builder definierst du eigene Pflichtfragen für die Mandats-Qualifizierung (Rechtsschutz, Mandanten-Historie, etc.).
